# InvoiceFlow

**Smart Invoicing for Freelancers & Small Businesses** - Create professional invoices, track expenses, and get paid faster.

## Monetization Model
- **Free**: 5 invoices/month, basic expense tracking
- **Pro ($19/mo)**: Unlimited invoices, expense categories, client management
- **Business ($49/mo)**: Recurring invoices, reports, multi-currency, API access

## Features
- Professional invoice creation with PDF export
- Expense tracking with categories & budgets
- Client management & payment tracking
- Dashboard with revenue analytics
- Stripe integration for payments
- Multi-currency support

## Tech Stack
### Web (Next.js)
- Next.js 14 + TypeScript + Tailwind CSS
- Prisma + SQLite/PostgreSQL
- Stripe for subscriptions
- Recharts for analytics

### Mobile (React Native)
- Expo + React Navigation
- Bottom tab navigation
- Native charts & gradients
- Push notifications

## Quick Start
```bash
# Web
cd web && npm install && npm run dev

# Mobile
cd mobile && npm install && expo start
```

## Ports
- Web: http://localhost:4010
- Mobile: Expo Go app
