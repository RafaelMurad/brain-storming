# Creative Framework App

A comprehensive web application to learn creative frameworks, understand creative flows, discover patterns, and boost your creativity with AI-powered ideation tools.

## Features

### 1. Creative Frameworks Library
Learn 10+ proven creativity methodologies including:
- **Design Thinking** - Human-centered approach to innovation
- **SCAMPER** - Seven thinking strategies for ideation
- **Six Thinking Hats** - Parallel thinking for decision making
- **Mind Mapping** - Visual technique for organizing ideas
- **Lateral Thinking** - Indirect approaches to problem solving
- **TRIZ** - Systematic innovation with 40 principles
- **Five Whys** - Root cause analysis technique
- **Classic Brainstorming** - Group creativity technique
- **Storyboarding** - Visual narrative planning
- **Random Input Technique** - Using random stimuli for ideas

### 2. Creative Flows
Understand how creativity works through different phases:
- **Divergent-Convergent Thinking** - The fundamental creative rhythm
- **Creative Problem Solving (CPS)** - Clarify, Ideate, Develop, Implement
- **Four Stages of Creativity** - Preparation, Incubation, Illumination, Verification
- **Flow State** - Achieving optimal creative performance

### 3. Creativity Patterns
Recognize and apply patterns that drive creative breakthroughs:
- Combination, Analogy, Inversion
- Constraint, Exaptation, Bisociation
- Serendipity, Provocation

### 4. AI-Powered Idea Generator
10 customizable prompt templates to use with AI assistants:
- Idea Expansion
- SCAMPER Analysis
- Problem Reframing
- Random Connection
- Future Scenario Planning
- Six Thinking Hats Analysis
- Analogy Generator
- Constraint-Based Creativity
- Five Whys Deep Dive
- Creative Brief Generator

### 5. Interactive Exercises
Practice creativity with guided exercises:
- Random Word Connection
- Assumption Reversal
- Worst Idea Brainstorm
- Constraint Challenge
- Analogy Hunt

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **Framer Motion** - Animations

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## How to Use AI Prompts

1. Go to the AI Idea Generator section
2. Select a prompt template that fits your need
3. Fill in the input fields with your challenge or idea
4. Click "Generate AI Prompt"
5. Copy the generated prompt
6. Paste into ChatGPT, Claude, or your preferred AI assistant
7. Review the AI's response and iterate on interesting ideas
