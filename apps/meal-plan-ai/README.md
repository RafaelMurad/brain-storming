# MealPlanAI

**AI-Powered Meal Planning** - Personalized recipes, smart grocery lists, and nutrition tracking.

## Monetization
- **Free**: 1 AI generation/week, basic grocery list
- **Premium ($9.99/mo)**: Unlimited AI generations, nutrition tracking, diet customization
- **Family ($14.99/mo)**: Multiple profiles, shared lists, family recipes

## Features
- AI-generated weekly meal plans
- Auto-generated grocery lists from meal plans
- Nutritional information per meal
- Dietary preference support (vegan, keto, etc.)
- Recipe instructions with video
- Smart pantry tracking

## Tech Stack
- **Web**: Next.js 14, OpenAI API, Tailwind
- **Mobile**: Expo, React Native
