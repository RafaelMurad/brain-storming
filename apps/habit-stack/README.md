# HabitStack

**Build Better Habits** - Track habits, build streaks, and transform your life.

## Monetization Model
- **Free**: 3 habits, basic tracking
- **Premium ($4.99/mo)**: Unlimited habits, detailed analytics, reminders, streaks
- **Lifetime ($49.99)**: All features forever

## Features
- Daily habit tracking with satisfying animations
- Streak counting with fire animations
- Week/month calendar views
- Statistics and progress charts
- Push notification reminders
- Habit categories and colors
- Export data

## Tech Stack
- **Web**: Next.js 14, Tailwind, Framer Motion
- **Mobile**: Expo, React Native, Haptic feedback
