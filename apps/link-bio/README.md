# LinkBio

**One Link for Everything** - Create a beautiful link-in-bio page for your social media.

## Monetization
- **Free**: 5 links, basic themes
- **Pro ($5/mo)**: Unlimited links, custom themes, analytics, QR code
- **Business ($15/mo)**: Custom domain, team access, priority support

## Features
- Drag-and-drop link ordering
- Multiple theme presets
- Click analytics per link
- QR code generator
- Social media icon integration
- Custom backgrounds
- Link scheduling

## Tech Stack
- **Web**: Next.js 14, Framer Motion, QRCode.react
- **Mobile**: Expo, React Native

## Ports
- Web: http://localhost:4013
