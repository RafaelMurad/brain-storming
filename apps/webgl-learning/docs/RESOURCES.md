# WebGL Learning Resources

A curated collection of official documentation, tutorials, courses, and tools to master WebGL.

## üìö Official Documentation

### WebGL Specifications & References

- **[Khronos WebGL Specification](https://www.khronos.org/webgl/)**
  - Official WebGL specification and reference
  - WebGL 1.0 and 2.0 specs
  - The authoritative source

- **[MDN WebGL API Documentation](https://developer.mozilla.org/en-US/docs/Web/API/WebGL_API)**
  - Comprehensive WebGL API reference
  - Tutorials and examples
  - Browser compatibility information

- **[WebGL Reference Card](https://www.khronos.org/files/webgl/webgl-reference-card-1_0.pdf)**
  - Quick reference PDF
  - All WebGL functions and constants
  - GLSL reference included

### GLSL (Shader Language)

- **[OpenGL ES Shading Language Specification](https://www.khronos.org/registry/OpenGL/specs/es/2.0/GLSL_ES_Specification_1.00.pdf)**
  - Official GLSL ES 1.0 spec (for WebGL 1.0)
  - Complete language reference

- **[GLSL Reference Guide](https://www.khronos.org/files/opengles_shading_language.pdf)**
  - Quick reference for GLSL
  - Built-in functions and variables

## üõ†Ô∏è Tools & Libraries

### Development Tools

- **[WebGL Inspector](https://benvanik.github.io/WebGL-Inspector/)**
  - Debug WebGL applications
  - Capture frames and inspect state
  - Chrome extension

- **[Spector.js](https://spector.babylonjs.com/)**
  - WebGL debugger and profiler
  - Capture and analyze frames
  - Performance analysis

- **[Shadertoy VSCode Extension](https://marketplace.visualstudio.com/items?itemName=stevensona.shader-toy)**
  - Live shader preview in VS Code
  - GLSL syntax highlighting
  - Auto-completion

### Libraries (For After Learning WebGL)

- **[Three.js](https://threejs.org/)**
  - Most popular WebGL library
  - High-level 3D graphics
  - Extensive examples and docs

- **[Babylon.js](https://www.babylonjs.com/)**
  - Game engine for web
  - Great documentation
  - Built-in physics and features

- **[PixiJS](https://pixijs.com/)**
  - 2D WebGL renderer
  - Fast and lightweight
  - Game and interactive graphics

- **[regl](http://regl.party/)**
  - Functional WebGL wrapper
  - Cleaner API than raw WebGL
  - Good middle ground

## üì± Additional Learning Paths

### After This Project

- **Web Audio API** - Combine with WebGL for audiovisual experiences
- **WebAssembly** - Performance-critical code
